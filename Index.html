Consejos para el uso de la aplicacion

Para comenzar a utilizar la aplicacion debe tener un archivo con extension .txt (nombre_del_archivo.txt). A este archivo lo denominaremos archivo de salida. Le sugerimos que el nombre del archivo sea lo mas descriptivo posible para Ud. Por ejemplo: Informatica2do.txt.

El contenido del archivo de entrada debe ir de la siguiente forma:

nombre_y_apellidos_1erEstudiante, nombre_y_apellidos_2do_Estudiante, ...

Nota: le sugerimos que preceda el primer nombre con un espacio, para mayor uniformidad.

El fichero de salida es un fichero pdf, su nombre sera el mismo que el del fichero de entrada pero seguido con la fecha del dia: nombre_del_archivo_dia_mes_año.pdf.

La aplicacion se encuentra en fase de prueba, cualquier sugerencia sera bien aceptada.
